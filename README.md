# LLM Wiki

**Persistent personal knowledge bases powered by LLMs.**

Traditional RAG rediscovers knowledge every single query.  
This project implements the **LLM Wiki pattern**: the LLM acts as a knowledge engineer that builds and maintains a clean, evolving Markdown knowledge base.

**Goal**: Turn your documents into a living second brain that accumulates refined knowledge over time instead of starting from scratch.

### Quick Start
1. Add your documents/notes/papers to the `raw/` folder
2. Paste `llm-wiki.md` into your LLM (Claude, GPT, Grok, etc.)
3. Tell the LLM to build and maintain the `wiki/` folder
