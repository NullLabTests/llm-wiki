# LLM Wiki
**Persistent personal knowledge bases powered by LLMs — beyond RAG.**

Traditional RAG rediscovers knowledge every query.  
**LLM Wiki** turns your LLM into a knowledge engineer that builds & maintains a clean, interlinked, evolving Markdown second brain.

**Goal**: Accumulate refined knowledge instead of starting from scratch.

### Quick Start
1. Drop docs into `raw/`  
2. Paste `llm-wiki.md` into any LLM  
3. Say: “You are my LLM Wiki engineer. Build and maintain the wiki from raw/”
